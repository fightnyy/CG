CANVAS
        In order to write on canvas, canvas must be transparent first.
        Otherwise, it'll all be covered and the writing won't be visible. 
        I made it transparent for the first time, but if it's completely transparent, 
        you won't know how far it's canvas and how much it's general space, 
        so I put canvas opacity Button!
        If you make it completely transparent, it comes out white!
        To make the canvas transparent,you should use gl.clearColor(0.0, 0.0, 0.0, a); 
        in here a is alpha value.
        If you set the alpha value to zero, you'll get a full transparency.
        And if you want the cube to be transparent, make a less than 1 in 
        the alpha value of vertexData.
        Also if you Enable the blend the color of the cube will be blended!

        i.e ; gl.enable(gl.BLEND);
Velocity
        The cube rotates only on the y-axis.
        Press the Upward key to accelerate the speed. When you start, you add 0.01 more. 
        Press the Upward key to accelerate in this way: 0.02 , 0.03 Of course,
        pressing the down direction key will reduce the speed.
        I divided the case of speed into zero and higher than zero.

CSS
        Even if the canvas is transparent,
        the letters cannot be placed inside the canvas so that they do not overlap.
        However, making position absolute and using z-index elements for css can solve this problem.
